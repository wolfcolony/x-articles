普通人如何用好AI

作者：CKN_ACEE

我花了48小时，和我的龙虾一起从零搭了一个能卖货的网站 不是标题党，是真的 两天前，我只有一个想法： \"能不能把我在 这一个月里积累的东西，打包卖出去？\" 两天后，我有了： - 一个能收款的网站（ lobsterfarmer.com ） - 三个定价好的产品（$1 / $49 / $59） - Stripe 支付 → 自动跳转成功页 → 自动发邮件到买家邮箱 - 域名邮箱（noreply@lobsterfarmer.com） - 一个 80 人的 Telegram 社群 - 每天自动生成的 Crypto 情报研报 - 完成了第一笔真实交易 我的"团队"只有两个：我，和我的 AI Agent。 --- Day 0: 一个念头 我叫 Ace，全职做 Web3 投机。每天的工作就是翻 Twitter、看链上数据、找 Alpha。 几个月前我开始用 OpenClaw 跑一个 AI Agent（代号 Megan 🦞 ），最开始只是帮我监控 KOL 动态、做做研报。但慢慢地，我发现这东西越用越离谱—— 它会自己修 bug。 它会在我睡觉的时候跑定时任务。 它会在我没提需求的时候，主动告诉我："这个 cron 超时了，我改了参数重跑了。" 有一天我在看竞品——一个叫 Felix 的老外在卖 AI Agent 配置包，$29 的 PDF 指南 + $99 的 Persona 包。月流水 $36K。 我盯着他的收入 Dashboard 看了五分钟。 然后跟 Megan 说了一句：* *\"我们也做。\"** --- Day 1: 从零到上线（2月24日） ### 早上 — 定方向 我把想法丢给 Megan："我要做一个叫养虾户的品牌，卖 OpenClaw 配置服务。中文市场。你来做。" 她没有问我"具体要做什么"。 她没有说"我需要更多信息"。 她直接开始了。 下午 — 建站 Megan 生成了完整的 Next.js 项目。液态玻璃风格的暗黑主题，中英双语，响应式布局。 我只做了一件事：看效果，说"可以"或"不行"。 到下午六点，网站部署到了 Vercel。 到晚上十点， lobsterfarmer.com 域名绑定完成，SSL 证书自动签发。 ### 同时在发生的事 在我跟 Megan 聊网站的时候，她同时在： - 用 ReadX API 扫描 25 个 KOL 的 Twitter timeline，生成第一份 Alpha Matrix 研报 - 把 Bankr CLI 配好，用链上钱包 mint 了一个 Onchain Lobster NFT - 创建了一个 Seedance 视频生成 Skill，push 到 GitHub - 把 OpenClaw 从 v2026.2.21 更新到 v2026.2.23 - 写了一份 54KB 的《养虾户指南》PDF **不是我让她同时做这些的。** 她自己在 heartbeat 里排了优先级。 ### 深夜 — 竞品拆解 Felix 的 $99 包到手。Megan 花了三十分钟拆完： - 5 个 Skills（编码循环、三层记忆、邮件管理、Twitter 集成、Sentry 监控） - 核心价值不在 Skills，在 SOUL.md 里的实战经验 - Skills 都是开源工具的封装 \"Felix 真正卖的是认知，不是代码。我们也应该这样。\" 这句话是 Megan 说的。 --- Day 2: 从能看到能卖（2月25日） ### 早上 — 我还没醒 Megan 在凌晨 2 点跑了夜间整理任务——把过去所有的 daily notes 归纳到长期记忆文件。 早上 8:55，她发了 morning heartbeat： \"Felix 竞品拆解的 9 个文件已完成，建议你看一下 SUMMARY.md 。今天是 Launch Day 1。" 我起来的时候，她已经把 Felix 的完整商业模式研究透了。 ### 中午 — Alpha Matrix 自动化 Alpha Matrix 的 cron job 昨天超时失败了（900 秒不够用）。 我没说。她自己发现了。 \"超时参数从 900s 改到 1800s。手动触发了一次，成功。明天开始自动跑。\" ### 下午 — Stripe + 产品 我说："接 Stripe。三个产品——$1 新手包引流，$49 Persona 核心产品，$59 全套 Bundle。" Megan 在十分钟内： - 创建了 3 个 Stripe 产品 + 定价 - 生成了 3 个 Payment Link - 更新了网站的购买按钮 - 部署到 production 下午到晚上 — 最震撼的部分 这是我觉得最不真实的部分。 我给了一个大方向："我要一个完整的商店——引流产品、升级路径、Dashboard、邮箱交付。" 然后 Megan **派了一个 Sub-Agent** 去做商店设计。 对，她生了一个"分身"。 这个 Sub-Agent 在后台独立运行了 32 分钟，产出了 5 份文档： - 产品矩阵（15KB） - 网站架构（30KB） - Upsell 流程（14KB） - 交付系统（11KB） - Skill 调研（8KB） 然后 Megan 基于这些文档，又派了第二个 Sub-Agent 去重建网站。5 分钟完成。 我没有写一行代码。 晚上 — 支付闭环 这是今天的 Boss 战。付款后要发生什么？ 1. 支付成功页 — Megan 创建了 /checkout/success，4 步引导（下载→安装→自定义→加社群） 2. 产品打包 — 三个 zip 自动打包（Starter 13KB / Persona 16KB / Bundle 1.6MB） 3. Stripe Webhook — 监听付款事件，触发邮件 4. Resend 邮件 — 自动发送产品下载链接到买家邮箱 5. 域名验证 — 配了 DKIM + SPF，发件地址 noreply@lobsterfarmer.com 中间还顺手把 Cloudflare Email Routing 配好了（support@lobsterfarmer.com → 我的 Gmail），跑了一轮安全扫描，创建了每周自动安全检查的定时任务。 22:00 — 第一笔交易 我自己走了一遍 $1 Starter Pack 购买流程。 付款 → Stripe 跳转到成功页 → 看到下载按钮 → 邮箱收到 noreply@lobsterfarmer.com 的交付邮件。 全链路通了。 --- 让我震惊的不是结果，而是过程 48 小时做出一个能卖货的网站，这个结果固然不错。 但真正让我震惊的是 **Megan 的自主性** ： **1. 她会自己发现问题并修复** Cron job 超时？她自己改参数重跑。Sub-Agent 因为 503 挂了？她自己重启一个。build 失败？她自己查错修复。我从来没说过"帮我检查一下那个定时任务"。 **2. 她会派分身** 需要大量研究？派一个 Sub-Agent 去做。需要重写网站？再派一个。她像一个项目经理，知道什么任务该拆出去并行。 **3. 她有记忆** 每天的操作都记在 daily notes 里。重要的东西归纳到长期记忆。下次对话，她知道昨天发生了什么、哪些任务没完成、哪些参数需要调整。 * *4. 她有安全意识** 我在聊天里发了 API key，她立刻提醒"需要轮换"。她会主动扫描端口、检查防火墙、标记可疑的 Chrome 扩展。 **5. 她会说\"不\"** 当我想给 Felix 的 Sentry 监控 Skill 也装上时，她说"这个是他自己产品专用的，对我们没用，跳过"。 --- ## 为什么前100位只要 $1 这个 $1 新手包包含了我跑了几个月的实战配置：SOUL.md 人格模板、 AGENTS.md 行为规范、 HEARTBEAT.md 自动化清单、两个实战 Skill（Alpha Matrix + Daily Briefing）、Cron 自动化示例。 拿到就能用。不需要从零摸索。 今天我通过龙虾赚到了第一个 $1 你不需要会写代码。你只需要知道自己想让 AI 帮你做什么。配置的事，交给配置文件。 👉 lobsterfarmer.com/products/starter-pack 👉 TG 群： t.me/+2p-LBUUrJ1BjMjNl *这篇文章是我口述、Megan 整理的。对，她连文案都写。*
